Cargos-containers

Les cargos-containers peuvent être composés d’un, de deux ou de trois containers. Ils permettent de stocker les marchandises que vous trouverez sur votre route.

Bien entendu, les marchandises revendues vous permettront de gagner beaucoup de Crédits, le but principal de votre vol. Vous devez donc ajouter à votre vaisseau le maximum de cargos-containers.

Cargos-containers spéciaux

Les marchandises rouges se revendent plus cher, mais elles peuvent uniquement être transportées dans des cargo-containers renforcés – les containers rouges. Les cargos-containers rouges ne contiennent souvent qu’un ou deux containers. La bonne nouvelle c’est qu’un cargo-container rouge peut accueillir n’importe quelle marchandise, quelle que soit sa couleur.

Les marchandises rouges ne peuvent pas être placées dans les cargo-containers blancs. N’importe quelle marchandise peut être placée dans un cargo-container rouge.

Les marchandises rouges se revendent plus cher, il est donc important d’ajouter un maximum de cargos-containers rouges à votre vaisseau.
