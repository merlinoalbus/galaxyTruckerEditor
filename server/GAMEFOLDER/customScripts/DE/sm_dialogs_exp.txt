SCRIPT ship_random_Dlg

  ShowDlgScene
  ChangeChar tutor campaign/tutor.png
  ShowChar tutor left
  Delay 400
  Say "Sorry, derzeit mache ich keine Tutorials."
  Delay 500
  ChangeChar tutor campaign/tutor-pizza.png
  Delay 500
  Ask "Ich hab grad Mittagspause."
  MENU
    OPT_IF tutor_somemissions "Erkennen Sie mich nicht?"
      ChangeChar tutor campaign/tutor-smile.png
      Ask "He! Der kleine Neuling! Was machst du denn hier?"
      MENU
        OPT "Ich wollte mir bloß mal diese Erweiterung ansehen."
          GO show_expansion_stuff
        END_OF_OPT
      END_OF_MENU
    END_OF_OPT
    OPT "Ich bin hier für ein Tutorial. Ich möchte das Zeug aus dieser Erweiterung kennen lernen."
      GO show_expansion_stuff
    END_OF_OPT
  END_OF_MENU

  LABEL show_expansion_stuff
  Delay 300
  ChangeChar tutor campaign/tutor-smile.png
  Delay 200
  Say "Oh, das wird dir gefallen."
  Say "Die Ingenieure der Corp mbH haben ein bahnbrechendes neues Schiffsteil entwickelt."
  Ask "Und sie möchten, dass du dieses unkonventionelle Schiffsdesign testest."

  LABEL ship_random_ask
  MENU
    OPT_IFNOT askedHowUnconventional "Äh... und wie unkonventionell ist dieses Schiffsdesign genau?"
      ChangeChar tutor campaign/tutor-smile.png
      ChangeChar tutor campaign/tutor.png
      Ask "Naja, so unkonventionell, dass du keine Ahnung hast, welche Form es haben wird, bevor du nicht mit dem Bau beginnst."
      SET askedHowUnconventional
      SET showOptCommentOnRandomDesign
      GO ship_random_ask
    END_OF_OPT
    OPT_IF showOptCommentOnRandomDesign "Sie wollen damit also sagen, dass dieses Design völlig zufällig ist, ja?"
      ChangeChar tutor campaign/tutor.png
      Say "Natürlich nicht!"
      Say "Es basiert auf einem ungeheuer hochentwickelten revolutionären Algorithmus, der den gravitonischen Druck der nächstgelegenen 18 Pulsare sowie die Quantenfluktuationen des Börsenmarkts berücksichtigt."
      Ask "Ein paar der schlausten Köpfe aus der Sanitärtechnik haben dran gearbeitet. Hat die ganze Nacht gedauert, bis es funktioniert hat."
      RESET showOptCommentOnRandomDesign
      GO ship_random_ask
    END_OF_OPT
    OPT_IFNOT askedTellNewComponent "Erzähl mir mehr über dieses \"revolutionäre\" neue Bauteil."
      ChangeChar tutor campaign/tutor.png
      Say "Es ist eine Ladefläche, die außerdem Batterien aufnehmen kann."
      ChangeChar tutor campaign/tutor.png
      Ask "... vielleicht ist es auch ein Batterieteil, das Waren aufnehmen kann."
      SET askedTellNewComponent
      SET showOptCommentOnCargoBattery
      GO ship_random_ask
    END_OF_OPT
    OPT_IF showOptCommentOnCargoBattery "Also ist dieses revolutionäre neue Bauteil bloß eine Ladefläche."
      ChangeChar tutor campaign/tutor.png
      Say "Es ist eine Waren- und Batteriefläche. Hast du nicht zugehört?"
      ChangeChar tutor campaign/tutor.png
      Ask "Es klingt vielleicht nicht besonders, aber du kannst damit mehr Zeug in dein Schiff stopfen. Und das ist immer gut."
      RESET showOptCommentOnCargoBattery
      GO ship_random_ask
    END_OF_OPT
    OPT "Hört sich gut an. Also los."
      ChangeChar tutor campaign/tutor-pizza.png
      Say "Viel Glück, Neuling."
      HideDlgScene
      RETURN
    END_OF_OPT
  END_OF_MENU
  HideDlgScene
  RETURN
  RETURN
END_OF_SCRIPT
