SCRIPT campaignIntro

  ShowDlgScene
  Delay 1000
  ShowChar tutor left
  Delay 1000
  RESET tutorialSeen
  IF_TUTORIAL_SEEN
    SET tutorialSeen
  END_OF_IF
  IFNOT tutorialSeen
    Say "Здорово, новичок. Я – твой инструктор.                                                                                                                                                    (Нажми для продолжения.)"
    Ask "Итак... что ты знаешь о космических грузоперевозках?                                                                                       (Выбери ответ.)"
  ELSE
    Say "Здорово, новичок. Я – твой инструктор."
    Ask "Итак... что ты знаешь о космических грузоперевозках?"
  END_OF_IF
  MENU
    OPT "Вообще ничего."
      Say "Не страшно."
      Say "Корпорация наняла меня как раз для того, чтобы я сделал из тебя первоклассного дальнобойщика."
      Ask "Надо только пройти обучение."
      SET playerIsBeginner
      EXIT_MENU
    END_OF_OPT
    OPT "Я играл в настолку, но цифровую версию еще не видел."
      Say "А, так тут все просто. Правила почти не изменились."
      Ask "Давай пройдем обучение."
      SET playerIsAdvanced
      EXIT_MENU
    END_OF_OPT
    OPT "Все знаю! Давай сразу к делу."
      Say "А... еще [g(один|одна)]."
      IF_TUTORIAL_SEEN
        Say "([g(Посмотрел|Посмотрела)] обучение и думает, что все знает.)"
      END_OF_IF
      SET playerIsCocky
      SET tutorialRefused
      Say "Лови космобус до космической станции Нубо-порт. Встретимся там."
      GO campaignIntroEnd
    END_OF_OPT
  END_OF_MENU
  MENU
    OPT "Круто. Давай!"
      Say "Хорошо."
      SET tutorialAsked
      EXIT_MENU
    END_OF_OPT
    OPT_IF tutorialSeen "Ну, обучалку-то я уже видел. Но можно посмотреть еще раз."
      Say "Верное решение."
      SET tutorialAsked
      EXIT_MENU
    END_OF_OPT
    OPT_IF tutorialSeen "Спасибо, но обучалку я уже [g(прошел|прошла)]."
      Say "Ну и ладно."
      Say "(То-то мне твое лицо знакомо.)"
      Say "Тогда вперед, на космобус до Нубо-порта, чо уж."
      SET tutorialSkipped
      GO campaignIntroEnd
    END_OF_OPT
    OPT_IFNOT tutorialSeen "Ненавижу обучалки. Давай сразу к делу."
      Say "[g(Уверен|Уверена)]?"
      Say "Это же самый верный способ научиться."
      Say "И мы вместе полетим до Нубо-порта."
      Ask "Будет круто."
      MENU
        OPT "Ладно, уболтал. Давай полетим."
          Say "Вот и славно."
          SET tutorialAsked
          EXIT_MENU
        END_OF_OPT
        OPT "Не, спасибо."
          SET tutorialRefused
          Say "Тебе же хуже."
          Say "Садись в космобус до Нубо-порта. Встретимся там."
          GO campaignIntroEnd
        END_OF_OPT
      END_OF_MENU
    END_OF_OPT
  END_OF_MENU

  LABEL campaignIntroEnd
  Delay 300
  HideChar tutor
  Delay 300
  HideDlgScene
  SUB_SCRIPT newbie_init
  IF tutorialAsked
    ACT_MISSION Tutorial
    SetCredits 0
  ELSE
    CenterMapByNode newbie
  END_OF_IF
END_OF_SCRIPT
  SCRIPT tutorialIntro
    ShowDlgScene
    Delay 500
    ShowChar tutor right
    Delay 500
    IF_FROM_CAMPAIGN
      Say "Добро пожаловать на склад Корпорации."
      Say "Здесь мы будем строить свои корабли."
    ELSE
      Say "Привет. Я буду твоим инструктором.                                                                                                                                                                                  (нажми для продолжения)"
      Ask "Готов[g(|a)] изучить основы космических грузоперевозок?                                                                                                           (выбери ответ)"
      MENU
        OPT "Так точно, сэр!"
          Say "Отличный настрой!"
          EXIT_MENU
        END_OF_OPT
        OPT "Ну, допустим."
          Say "Больше жизни! Ты же будешь строить настоящий космический корабль!"
          EXIT_MENU
        END_OF_OPT
        OPT "Э-э-э... чо-т это вообще не по мне. Можно выйти?"
          Say "*Вздох*"
          Delay 300
          Delay 300
          Say "Эй, парни, еще [g(один|одна)]. Вышвырните [g(его|ее)]."
          HideChar tutor
          HideDlgScene
          QuitCampaign
          RETURN
        END_OF_OPT
      END_OF_MENU
    END_OF_IF
    Delay 300
    ChangeChar tutor campaign/tutor-out.png
    Delay 400
    Say "Эй, профессор, доставай свои слайды!"
    Delay 300
    ChangeChar tutor campaign/tutor.png
    Delay 300
    HideChar tutor
    HideDlgScene

    AddInfoWindow tutorial/infoGameGoal2.png
    AddInfoWindow tutorial/infoGameGoal3.png
    AddInfoWindow tutorial/infoGameGoal4.png

    Say "Я буду наблюдать за тобой на тот случай, если тебе понадобится помощь."
    Delay 300
    HideChar tutor
    Delay 300
    HideDlgScene
  END_OF_SCRIPT
